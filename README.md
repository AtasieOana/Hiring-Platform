### Colors:
The **60-30-10** rule is used:
* primary color (occupies 60% of the page): white
* secondary color (occupies 30% of the page): beige (#f9cbc1, #fff0db, #fbc094)
* accent color (occupies 10% of the page): green (#87ae9a)
